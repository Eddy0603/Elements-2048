Element 2048


The game works by the player using the arrow keys to move blocks with element symbols on them, combining the elements, forming the next element. The objective of the game is to reach Calcium (Ca).

As the player “discovers” a new element by merging two of the previous elements, it shows the player some information about the newly discovered element.

Players can compete with each other by comparing high scores, the real symbol of skill.
